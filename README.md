# <a href="https://wholesome-cats.vercel.app">Wholesome-Cats</a>

## Description:

- Simple project to practice using react hooks, typescript and error handling
- Fetches a random cat image using the CATAAS (Cat as a service) API

### User will be able to:

- Generate a random cat image at the click of a button

## Technologies Used:

### Frontend

- Vite + React
- Typescript
- HTML5/CSS3
- Axios

### Backend
- Node.js
- Express

## Installation and Setup

Clone down this repository. You will need `node` and `npm` installed globally on your machine.  

Installation:

#### Install Dependencies
Frontend (root directory)
`npm install`

Backend
`cd backend`
`npm install`

#### Run the Backend Server

From the /backend directory:

`node app.js`

#### Run the Frontend Server

Open a new terminal, go back to the root directory, and run:

`cd ..`
`npm run dev`

#### Access the Application

Frontend:

http://localhost:5173

Backend:

http://localhost:5000
